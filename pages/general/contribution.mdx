## Seiten

Jede Seite besteht aus einer [MDX-Datei](https://mdxjs.com/), die man im
Ordner `pages/` finden kann. MDX-Dateien sind im Grunde nur Markdown, in
denen React-Komponente sowie Html-Tags direkt genutzt werden können. Eine
mögliche Seite könnte so aussehen:

````mdx filename="pages/general/meine-seite.mdx"
# Seite Überschrift

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Cras sagittis malesuada
risus at volutpat. Pellentesque eget pretium diam. Maecenas lobortis gravida 
enim, non lobortis sapien maximus ut.

```py
a = 1
b = 2
c = a + b
```

## Abschnitt 1

```math
\pmatrix{
  1 & 2 & 3 \\
  4 & 5 & 6 \\
  7 & 8 & 9
}
```

## Abschnitt 2

Verweise auf [andere Websiten](https://bwinf.de) sind ebenfalls möglich. 
````


## React-Komponenten

In MDX kann man React-Komponenten ohne Weiteres direkt verwenden, z.B.: 

```mdx filename="pages/general/meine-seite.mdx"
import IncompleteSection from '@components/IncompleteSection.tsx'


# Seite Überschrift

<IncompleteSection />
```

## Literaturverweis

Man kann weiterführende Literatur bzw. Quelle für eine Seite angeben:

```mdx filename="pages/general/meine-seite.mdx"
import { ReferenceList, ReferenceListItem } from '@components/ReferenceList.tsx'

<ReferenceList>
    <ReferenceListItem url="https://seite1.de" urlText="Seite 1" desc="Beschreibung der Quelle."></ReferenceListItem>
    <ReferenceListItem url="seite2.de" urlText="Seite 2" desc="Beschreibung"></ReferenceListItem>
    <ReferenceListItem url="seite3.de" urlText="Seite 3" desc="Beschreibung"></ReferenceListItem>
    <ReferenceListItem url="seite4.de" urlText="Seite 4" desc="Beschreibung"></ReferenceListItem>
</ReferenceList>
```


## Ordnerstrukturen

Das Routing der Website folgt dem Pfad der MDX-Dateien in `pages/`. So
wird z.B. die Datei `pages/general/contribution.mdx` unter der URL
`/general/contribution` erreichbar sein.

Es ist möglich, die Überschrift der jeweiligen Seiten zu einem anderen als
dem eigenen Datei-Namen zu ändern. Dazu fügt man einen Eintrag in der
Datei `pages/**/_meta.json` wie folgt hinzu:

```json filename="pages/general/_meta.json"
{
  "index": {
    "title": "Willkommen"
  },
  "contribution": {
    // NEU: 
    "title":"Beitragen"
  }
}
```

Mehr dazu findet man unter [Nextra - Page Configuration](https://nextra.site/docs/docs-theme/page-configuration).


## Tech-Stack

Die Website basiert auf [Nextra](https://nextra.site/), einem
React-Framework für die Erstellung von statischen Websites. Nextra nutzt
[Next.js](https://nextjs.org/) und [MDX](https://mdxjs.com/). Für CSS nutzen wir [Tailwind](https://tailwindcss.com/).
